# 📚 Online Bookshop – E-Commerce Website

A responsive e-commerce website developed as part of my first year Computer Science coursework at Manchester Metropolitan University.

The project simulates an online bookstore where users can browse products, enter payment details, and complete a checkout process. It demonstrates responsive web design, client-side form validation, and interaction with a REST API.

---

## Features

- 📱 Responsive layout for desktop and mobile devices
- 🛒 Browse books
- 💳 Secure client-side payment validation
- ✅ Real-time validation error messages
- 🔗 REST API integration using Fetch API
- 💾 Session storage used to transfer checkout information securely
- ⚠️ Success and failure handling during checkout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- REST API
- Session Storage
